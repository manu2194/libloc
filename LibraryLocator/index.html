Libraries Near You

Our Purpose

This website was created to help residents of Prince George's County, which includes parents, teens, college students, and others, find a library near them. Our goal is to solve the problem that these people face, which is a lack of computer access or reliable internet access at home. We created an interactive map that pinpoints different libraries in the county based on the location of the user and the location of possible destinations.
